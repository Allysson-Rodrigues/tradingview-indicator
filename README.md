# ICT + FVG + RSI (TradingView)

Indicador avançado desenvolvido em **Pine Script v6** para TradingView, combinando:
- ICT HTF Candles
- Fair Value Gaps (FVG)
- RSI Multi-Timeframe
- EMA
- Alertas e painel visual

## Recursos
- HTF Candles até Weekly
- Detecção de FVG com mitigação configurável
- RSI MTF com tabela dinâmica
- Threshold adaptativo por timeframe ou ATR
- Alertas configuráveis
- Otimizado para performance

## Como usar no TradingView
1. Abra o TradingView
2. Vá em **Pine Editor**
3. Copie o conteúdo de:
4. Cole no editor
5. Clique em **Add to Chart**
